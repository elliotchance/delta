levenshtein
Calculate Levenshtein distance between two strings.

Syntax

int levenshtein ( string str1 , string str2 )

Description

The Levenshtein distance is defined as the minimal number of characters you have to replace, insert or delete to transform str1 into str2. The complexity of the algorithm is O(m*n), where n and m are the length of str1 and str2.

Parameters

str1 One of the strings being evaluated for Levenshtein distance.
str2 One of the strings being evaluated for Levenshtein distance.

Return Values

This function returns the Levenshtein-Distance between the two argument strings.
