expm1
Returns exp(number) - 1, computed in a way that is accurate even when the value of number

Syntax

number expm1 ( number arg )

Description

is close to zero. expm1() returns the equivalent to 'exp(arg) - 1' computed in a way that is accurate even if the value of arg is near zero, a case where 'exp (arg) - 1' would be inaccurate due to subtraction of two numbers that are nearly equal.

Parameters

arg The argument to process.

Return Values

'e' to the power of arg minus one.
