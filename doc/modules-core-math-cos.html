cos
Cosine.

Syntax

number cos ( number arg )

Description

cos() returns the cosine of the arg parameter. The arg parameter is in radians.

Parameters

arg An angle in radians.

Return Values

The cosine of arg.

Examples

Example #1 cos() example 
println(cos(M_PI));

See Also

  • acos() - Arc cosine.
  • sin() - Sine.
  • tan() - Tangent.
