feof
Tests for end-of-file on a file pointer.

Syntax

boolean feof ( resource handle )

Description

Tests for end-of-file on a file pointer. fopen() or fsockopen() (and not yet closed by fclose()). otherwise returns FALSE.

Parameters

handle The file pointer must be valid, and must point to a file successfully opened by

Return Values

Returns TRUE if the file pointer is at EOF or an error occurs (including socket timeout);
