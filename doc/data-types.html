Delta Documentation

Data types

Overview

Delta like all scripting language is typeless, allowing any data type to be assigned to any variable and also change at any time. This has its advantages and disadvantages, but all scripting languages will convert types when necessary.

Internally Delta will keep the following types:

  • NULL
  • BOOLEAN
  • NUMBER
  • STRING
  • ARRAY
  • OBJECT
  • RESOURCE
