Delta Documentation

Macros

Overview

This is a list of the macros that a developer may need to know.

Delta Functions

Macro Value Description
DELTA_ARG(i) Get the value of a function argument.
DELTA_ARG0 - DELTA_ARG9 The same as DELTA_ARG(0) - DELTA_ARG(9).
DELTA_ARG0_NAME - DELTA_ARG9_NAME The same as DELTA_ARG_NAME(0) - DELTA_ARG_NAME(9).
DELTA_ARGS Returns the number of arguments sent to this function as an int.
DELTA_ARG_NAME(i) Get the name of the argument key. This is returned as a null-terminated string regardless of the type.
DELTA_CHECK_RESOURCE(resource, type) Check the variable resource is a RESOURCE and that its type is also equal to type.
DELTA_DEST The destination variable, this is the variable that holds the return value. You should use the DELTA_RETURN_* macros instead of accessing this directly.
DELTA_ENUMERATE_ARRAY Enumerate the keys and values of an array.
DELTA_FALSE 0 The constant for the delta value of false.
DELTA_FUNCTION Define a delta function body.
DELTA_RELEASE(obj_release, obj_ptr) Release the variable obj_ptr if obj_release is not equal to zero. This is a easier way of freeing memory returned by delta_cast_string().
DELTA_RETURN_BINARY_STRING(return_value, return_size) Delta has binary safe strings (meaning strings can contain binary characters and the NUL character.) Not always but sometimes you may want to use DELTA_RETURN_BINARY_STRING to make sure the returned string is stored correctly.
DELTA_RETURN_BOOLEAN(return_value) Return the BOOLEAN value of return_value.
DELTA_RETURN_FALSE Return the BOOLEAN value of false.
DELTA_RETURN_NAN Return the NUMBER constant for Not A Number.
DELTA_RETURN_NULL Return NULL.
DELTA_RETURN_NUMBER(return_value) Return the NUMBER value of return_value.
DELTA_RETURN_ONE Return the NUMBER value of 1.
DELTA_RETURN_RESOURCE(return_value, resource_id) Return the binary resource return_value with the resource type of resource_id.
DELTA_RETURN_STRING(return_value) Return a null-terminated string.
DELTA_RETURN_TRUE Return the BOOLEAN value of true.
DELTA_RETURN_ZERO Return the NUMBER value of 0.
DELTA_TRUE 1 The constant for the delta value of true.

Constants

Macro Value Description
DELTA_SUCCESS 1 Successful (pass) result.
DELTA_FAILURE 0 Unsuccessful (fail) result.
DELTA_YES 1 Yes.
DELTA_NO 0 No.

Data Types

Macro Value Description
DELTA_TYPE_ARRAY 4 Data type is ARRAY.
DELTA_TYPE_BOOLEAN 1 Data type is BOOLEAN.
DELTA_TYPE_NULL 0 Data type is NULL.
DELTA_TYPE_NUMBER 2 Data type is NUMBER.
DELTA_TYPE_OBJECT 6 Data type is OBJECT.
DELTA_TYPE_RESOURCE 5 Data type is RESOURCE.
DELTA_TYPE_STRING 3 Data type is STRING.
DELTA_TYPE_UNKNOWN 10 This isn't actually a real data type but a theoretical place holder in rare situations.

Errors and Warnings

Macro Value Description
DELTA_ERROR_ERROR "1" An ERROR will stop the script from executing.
DELTA_ERROR_WARNING "2" A WARNING signals an error or failure that is not fatal.
DELTA_ERROR_PARSE "4"  
DELTA_ERROR_NOTICE "8"  
DELTA_ERROR_USER_ERROR "256"  
DELTA_ERROR_USER_WARNING "512"  
DELTA_ERROR_USER_NOTICE "1024"  
DELTA_TRIGGER_ERROR(msg, type) Trigger a runtime error to be dealt with by the VM. type is one of DELTA_ERROR_*.

Delta Modules

Macro Description
DELTA_PREPARE_MODULE_FUNCTIONS(total) Begin exporting functions.
DELTA_PUSH_FUNCTION(name, min, max) Export a function to the Delta VM.
DELTA_END_MODULE_FUNCTIONS End and clean up exporting functions.
