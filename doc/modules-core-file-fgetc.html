fgetc
Gets character from file pointer.

Syntax

string fgetc ( resource handle )

Description

This function forces a write of all buffered output to the resource pointed to by the file handle. fopen() or fsockopen() (and not yet closed by fclose()). Returns FALSE on EOF.

Parameters

handle The file pointer must be valid, and must point to a file successfully opened by

Return Values

Returns a string containing a single character read from the file pointed to by handle.
