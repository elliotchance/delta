ctype_space
Check for whitespace character(s).

Syntax

bool ctype_space ( string text )

Description

Checks if all of the characters in the provided string , text, creates whitespace. otherwise. Besides the blank character this also includes tab, vertical tab, line feed, carriage return and form feed characters.

Parameters

text The tested string.

Return Values

Returns TRUE if every character in text creates some sort of white space, FALSE
