lcg_value
Random number between 0 and 1.

Syntax

number lcg_value ( )

Description

The true lcg_value() returns a pseudo random number in the range of (0, 1), combining two CGs with periods of 2^31 - 85 and 2^31 - 249. The period is equal to the product of both primes. However this implementation of the function only uses rand() and RAND_MAX to deliver a random number between 0 and 1.

Return Values

Random number between 0 and 1.

See Also

  • rand() - Generate a random integer.
  • mt_rand() - Generate a better random value.
