abs
Absolute value.
 

Description
Returns the absolute value of number. is also float , otherwise it is integer (as float usually has a bigger value range than integer).
 

Parameters
numberThe numeric value to process.
 

Return Values
The absolute value of number. If the argument number is of type float , the return type
 

Examples
@example Example #1 abs() example
$abs = abs(-4.2); // $abs = 4.2; (double/float)
$abs2 = abs(5); // $abs2 = 5; (integer)
$abs3 = abs(-5); // $abs3 = 5; (integer)
